# Data Structures and STL Containers
## Continers
- Container is an object that stores a collection of other objeccts
- Container manages the storage space for its elements & provides member functions to access them
    - Stack, Map, Vector etc 
    ![Arrary of flowers](images/ArrayOfFlowers.png)

1. **Sequence Conntainers** (Data stores linearly)
    - Array 
    - Vector
    - Deque
    - Forward List
    - List

2. **Associative Containers**
    - Set (Store only unique, ordered way)
    - Map (Collection of Key value pair, sorted by key)
    - Multiset
    - Multimap

3. **Unordered Assocoative Containers**
    - Unordered Set
    - Unordered Map
    - Unordered Multiset
    - Unordered Multimap

4. **Containers Adaptors**
    - Stack
    - Queue 
    - Priority Queue 

### Common operations in Containers
- Insertion 
- Searching 
- Deletion
